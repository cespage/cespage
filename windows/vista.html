Windows Vista Home Basic Package

Easier

With the Windows Vista™ operating system, it's easier and faster than ever to find, use, manage, and share the information on your PC or on the Web.

Instant Search

Simply type in a word contained within a document or e-mail message, the artist of a song, or the date a picture was taken, and Instant Search will bring back any matches.

Photo Gallery

Organise a lifetime of memories with ease using Windows Photo Gallery. By tagging your photos by date, keyword, star rating, or any identifying label you choose, you can find them when you want them.

Safer

Relax. With the Windows Vista™ operating system, the security and health of your PC is automated across everything you do, so you can focus on the things you enjoy.

Internet Explorer 7

The new Windows® Internet Explorer® 7 helps protect your PC and your personal information against malicious software, fraudulent Web sites, and online phishing scams.

Defender

Windows Defender helps defend your PC against pop-ups, slow performance, and security threats caused by spyware and other unwanted software.

More Entertaining

Expand your ability to enjoy and share your television, photos, music, and games with the Windows Vista™ operating system.

Games

It's easier for you to find, play, and manage your games with Games Explorer, which provides detailed information including when you last played, genre, and rating of your games. With Microsoft DirectX® 10, play vivid and engaging games and use the same game controller with both your PC and your Xbox 360™ system.

More Mobile

Set up a wireless network at home with Network and Sharing Center - so you can experience the freedom of working virtually anywhere in your home. Then find and join a wireless network at your favorite hotspot to help you stay productive wherever you go.

Remote Access

Access information, collaborate with others, and exchange data from a variety of locations with the Windows Vista™ operating system. Now you can connect to your office, wherever the road takes you.

Features

Security - User Account Protection enables administrators to give end users only the privileges they need to perform their tasks. Bridging the gap between user and administrative privileges by running applications with limited permissions. Windows Service Hardening monitors critical services for abnormal activity, registry and network that could allow malware plus other Anti-malware features detect and remove worms, viruses and other malicious software. Advanced data protection technologies reduce the risk that data will be viewed by unauthorised users.

Deployment - Windows Imaging format provides a single file that contains complete installation images. Windows Pre-installation Environment enables administrators to configure Windows before launching the setup process. The Application Compatibility Toolkit helps administrators identify and resolve any issues with applications being migrated.

Manageability - Auto-diagnosis and auto-correction of common error conditions, fixes for known crashes and technology to minimise reboots when installing software. Improved Task Scheduler enables tasks to launch when a specific event occurs. Web Services for Management makes it easier to run scripts remotely and to perform other management tasks. Microsoft Management Console 3.0 provides a common framework for management tools.

Performance - Startup software process in the background while users perform their desired tasks on system startup. The new Sleep state combines the speed of Standby mode with data protection features and low-power consumption of Hibernate. Superior memory management and improved input/output management improves responsiveness, especially in tasks, such as opening the Start menu or right-clicking a file in Windows Explorer.
