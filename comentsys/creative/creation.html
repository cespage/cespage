Creation

Comentsys Creative can create Website, Graphic and Software creations for businesses and individuals for many different purposes and uses, from simple one-page websites, company logos, through to custom software creation.

Comentsys Creative can help your business communicate with the world with our Website Creation service, offering One-page website for small businesses and individuals, through to template designs find your favourite website template and we'll customise it for your use, or a custom website with a unique look and feel for your business or personal use.

Comentsys Creative can help brand your business or visualise an idea with out Graphic Creation service, offering logo design for a business or product, plus Advertising and Business card design is also available, also there is Powerpoint/presentation, CD-ROM and other interactive graphic design available too.

Comentsys Creative can help empower your business or give you the solution you need with Software Creation, offering white-label products which is software created that can be branded, plus custom software creation allows for the creation of a bespoke application.

Website Methodology

Comentsys Creative can employ a variety of web standards for your website such as XHTML 1.0, HTML 4.01, CSS and can label your site for ICRA Standards also.

Websites are created using HTML-Kit, a Text-based HTML Editor giving maximum flexability plus includes checking systems for web standards, we can also use visual tools however this applicatyion allows for optimium file size and load time.

Website Technologies Supported:

  • HTML 3.2 and 4.1
  • XHTML 1.0
  • CSS 1.0 and 2.0
Graphic Methodology

Comentsys Creative uses Paint Shop Pro 7 to create all main graphics for Website and Graphic creation. Graphics can be created as a vector which allows resizing without loss of image quality and resolution, and raster or Bitmap style graphics can be created like Photos or set sized images. We can also make use of Macromedia Flash for graphics, animation and interative elements.

Graphics Technologies Supported:

  • Bitmaps : GIF, PNG, BMP, JPEG et al.
  • Vector : PSP, EMF, WMF
  • Flash 6
Software Methodology

Comentsys Creative uses Microsoft .NET 2.0 in White Label and Custom Software creation, programmed using Visual Basic 2005 Express, utilising the latest in software creation technology. Software can also be created using Visual Basic 6 for environments where .NET is not supported.

Software Technologies Supported:

  • .NET 2.0
  • Visual Basic 2005 and Visual Basic 6
  • ASP.NET and ASP
