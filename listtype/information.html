Product

ListType.com is the Multi-item List Add-on from CESWare.com, the software solutions part of Comentsys.
Allows items from a list to be inserted into a document, for example standard text like an address for a letter or something that would be typed over and over again like a signature for a document. These items can be managed using ListType.com, plus it also supports other type of content such as custom objects like images or almost any collection of information. Items can be easily added or inserted to and from the list and a list manager is also available to view items in the list.
