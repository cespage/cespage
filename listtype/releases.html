Integrated

ListType.com Integrated is the component based release of ListType.com, allowing third-part developers to include this in their applications, it is this component that features in all releases of CESWord.com. Also included with this release are free button graphics, this component can be integrated into an applications own user interface or can supply its own floating toolbar if needed.

Solo

ListType.com Solo is the stand-alone release of ListType.com, this is the ListType.com Integrated release within an application, it supplies a special interface for this control, the application will allow you to manage clipboard items by adding them to ListType.com so you can cut-and-paste many things without needing to lose what was cut-and-pasted before.
