Integrated

KlikType.com Integrated is the component based release of KlikType.com, allowing third-party developers to include this in their applications, it is this component that also features in CESWord.com Standard. Also included with this release is the KlikType.com Developer Kit allowing keyboard layouts to be created and tested for uses with this and other KlikType.com enabled items.

Download : In Development

Solo

KlikType.com Solo is the stand-alone release of KlikType.com, this is the KlikType.com Integrated release in an application so the keyboard component itself is exactly the same, this release allows it to be used in conjunction with other applications. This release features the On-screen Keyboard with a text entry area where text can be typed using the KlikType.com keyboard then cut-and-pasted into another application.
