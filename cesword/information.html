Product

CESWord.com is the Word Processing software application from CESWare.com the software solutions part of Comentsys.
Using a Tab and Toolbar based interface, CESWord.com brings all the main Rich Text editing features to the application. These include Printing, Editing, Hyperlinks, Alignment, Colour, Highlighting, Text Styles, Indenting plus other features such as Word Count and ListType which can manage small pieces of insertable text. All these features are in all editions of CESWord.com, CESWord.com Compact comes with all the mentioned features whereas CESWord.com Standard has a Multiple Document Interface plus KlikType.com support, the on-screen keyboard allowing text entry via the mouse!
