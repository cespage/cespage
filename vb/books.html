Murach's Visual Basic 6

Quality: This book is superb from beginner to advanced there is something to learn from this book, in a way that introduces you to new concepts and explains where necessary in a well constructed manner.
Ease of Use: If you are a beginner you'll find this book a must especially for ADO database programming but the book also covers other business applications.
Ability Level: This book is good for all levels, even if you have never used Visual Basic before you can start from the basics of how to use VB, and if you are already familiar with it you can simply skip those sections. This book covers all levels of programming using VB in a business.
Usability: This book is easy to use as there are full printed code examples to construct fully working applications and are in an easy to read format, properties and other concepts are explained well.
Usefulness: For business applications this book is a must, plus if you are just starting Visual Basic this should be the book to use. The book also covers ActiveX and other required development skills.
Overall: This book is so good it is ideal as a training manual for new users of Visual Basic, or those who have never done database programming as this book covers everything you'd need to know.

Rating 5 out of 5
