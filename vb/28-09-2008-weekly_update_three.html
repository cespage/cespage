Q&A

In this regular section of the Weekly Update: Q&A. This answers Visual Basic related questions asked via CESPage.com Blog and the website CESPage.com Visual Basic.

 With the startpage in webbrowser. I'm trying months to find out the code... without success... asked by Kevin

The Homepage for the WebBrowser control is determined by the settings in Internet Explorer (Tools, Internet Options, Homepage) if using the GoHome method. However you could have the browser navigate to another website using the Navigate function eg. cespage.com/vb when the application is loaded or when the Home option is selected. 

Navigate("http://www.cespage.com/vb")
