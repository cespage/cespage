Visual Basic 2008 Express Edition

With the release of Visual Basic 2008 Express Edition came a host of brand new features, some of which will feature in the new tutorials, such as WPF/XAML which is the basis of the first tutorial, Hello World, which uses this new technology instead of Windows Forms. All the games will be done as WPF/XAML applications in the future tutorials, however the other tutorials will still remain as Windows Forms so you can create these in both Visual Basic 2005 and Visual Basic 2008 (.NET 2.0/3.0,3.5).
