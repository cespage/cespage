Features

Notes - Selection of useful notes available including Desktop note for Rich Text Notes, Doodle note for basic drawing, Task note for to-do lists and Photo note for displaying your favourite photos, plus Web note for Internet browsing, Event note for appointments, Contact note for your home & business contacts and Launch note for shortcuts to applications and files.

Manage - See and created notes using Notes feature including filtering support for Flags and Note type plus Favourites. Pinboard allows you to send notes from the Desktop to the Pinboard which is a seperate window to contain these notes, with the ability to manage these notes such as view styles and more.

Customise - Change the colour scheme used on the toolbars and main toolbar easily from the Options, choose from various styles and also other settings such as what to do when the application starts and more! Plus Theme support for the main interface, customise it with new images and styles from those available or download new ones created by Developers.

Helpful - All notes support Flags for ease of reference set your home related Notes with a Green flag, and the Important ones with a Red flag, it's up to you! Plus you can mark a Note as a favourite making it easier to find in the Notes feature. You can even set an Alert to have a note appear at a particular date and time.
