Visual Web Developer 2010 Express for Silverlight Tutorials

CESPage.com Silverlight features tutorials for Silverlight created using Visual Web Developer 2010 Express Edition, these can also be done in Visual Studio 2010 also.
Suggest your idea on Twitter with the Hashtag #cpesl It can be any topic such as XAML, XML, LINQ or based on an existing example or application, as long as Silverlight supports it! Submitted ideas will be listed on the Tutorials page if they have been selected - there won't be any more tutorials unless more are suggested so get Tweeting!
